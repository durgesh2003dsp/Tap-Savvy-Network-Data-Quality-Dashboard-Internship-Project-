# Tap-Savvy-Network-Data-Quality-Dashboard-Internship-Project-
Built a Power BI Data Quality Dashboard for Tap Savvy Network to analyze profile completeness and missing customer information.

### **1. Project Overview**

This Power BI dashboard provides a complete evaluation of profile completeness, missing information, and company-wise customer status for Tap Savvy Network.
It helps identify data gaps, monitor active/inactive profiles, and highlight fields that require updates for improved data accuracy and business efficiency.

## **2. Key Objectives**

Track total profiles, active profiles, and inactive profiles.

Identify missing fields across the customer database.

Visualize company-wise profile status to prioritize corrections.

Measure data quality using percentages and counts.

Provide interactive filters for quick drill-down by location, company, and designation.

## **4. Tools & Techniques Used**

Power BI Desktop

DAX Measures for totals & percentages

Donut Charts, Bar Charts, KPIs, Pie Charts

Data Modeling for relationships

Interactive slicers and drill-down features

Custom dark theme UI for better visual contrast

## **5. Insights & Takeaways**

Over 1000 fields across profiles are missing important data.

Profile pictures, company descriptions, and website details are the most incomplete.

Several companies have high inactive-profile counts—priority areas for outreach.

Overall profile completeness needs improvement for accurate business communication.

Dashboard helps the team take actionable steps to improve data quality.
